## 🏗️ Tech Stack Summary

| Layer | Technology |
|-------|-----------|
| Frontend | React 18 + Tailwind CSS + Electron |
| Backend | FastAPI + Python 3.11+ |
| Database | PostgreSQL 15 (3NF+ normalized) |
| ORM | SQLAlchemy 2.0 + Alembic |
| AI/ML | XGBoost, LightGBM, Scikit-learn, Isolation Forest |
| Computer Vision | PyTorch + YOLOv8 (Ultralytics) + OpenCV |
| LLM | OpenAI GPT-4o-mini / Ollama (Llama 3) |
| Explainable AI | SHAP + LIME |
| Auth | JWT (RS256) + bcrypt |
| State Management | Zustand |
| Visualization | Recharts |
| Containerization | Docker + Docker Compose |
| CI/CD | GitHub Actions |
| Cloud | AWS (ECS, RDS, S3, CloudFront) |
